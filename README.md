## Sexual Health App

A JavaFX desktop application designed to improve access to sexual health information through evidence-based educational content, medication reminders, event management, and privacy-focused features. Developed using an Agile Scrum workflow as part of a six-person university software engineering team.

## Technology Stack
- Java 21
- JavaFX
- FXML
- Maven
- CSS
- Git & GitLab
- JUnit 5
- CommonMark (Markdown rendering)
- Agile Scrum
